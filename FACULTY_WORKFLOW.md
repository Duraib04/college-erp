# Faculty Workflow

This workflow is for daily faculty operations.

## Objective
Handle teaching, attendance, assessments, student support, and reporting.

## Steps

1. Login and Daily Review
- Open Faculty Dashboard
- Check timetable, notifications, and pending actions

2. Course Handling
- Go to Faculty -> Courses
- Confirm assigned courses and class lists

3. Attendance Update
- Go to Faculty -> Attendance
- Mark and save attendance by class/session

4. Assignment and Assessment
- Go to Faculty -> Assignments
- Publish assignments and track submissions
- Go to Faculty -> Grades
- Enter/update marks and feedback

5. Exam and Syllabus Tracking
- Go to Faculty -> Exams
- Update exam details/results
- Go to Faculty -> Syllabus
- Track completion progress

6. Student Support and Communication
- Go to Faculty -> Complaints
- Go to Faculty -> Notifications
- Resolve student issues and post updates

7. Documentation
- Go to Faculty -> Course Diary
- Maintain session-wise teaching record
- Go to Faculty -> Reports
- Review periodic teaching and class performance reports
