# Student Workflow

This workflow is for student portal usage.

## Objective
Use portal features for academics, communication, and self-tracking.

## Steps

1. Dashboard Check
- Open Student Dashboard
- Review announcements, attendance summary, and due tasks

2. Timetable and Courses
- Go to Student -> Timetable
- Go to Student -> Courses
- Follow daily class schedule and course resources

3. Attendance and Results
- Go to Student -> Attendance
- Monitor attendance percentage regularly
- Go to Student -> Results
- Track semester and internal performance

4. Assignments and Exams
- Go to Student -> Assignments
- Check deadlines and submit on time
- Go to Student -> Exams
- Review exam schedule and status

5. Fees, Library, and Certificates
- Go to Student -> Fees
- Track payment and dues
- Go to Student -> Library
- Track borrow/return status
- Go to Student -> Certificates
- Access available certificates/documents

6. Complaints and Notifications
- Go to Student -> Complaints
- Raise issues with clear details
- Go to Student -> Notifications
- Stay updated with institute announcements

7. Profile and Settings
- Go to Student -> Profile
- Keep personal/contact details up to date
- Go to Student -> Settings
- Maintain account preferences
