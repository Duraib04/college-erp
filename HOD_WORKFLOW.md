# HOD Workflow

This workflow is for department-level academic monitoring.

## Objective
Manage and review all department activities after admin setup.

## Steps

1. Department Overview
- Open HOD Dashboard
- Review department stats and pending tasks

2. Faculty Monitoring
- Go to HOD -> Faculty
- Review faculty list and engagement

3. Student Monitoring
- Go to HOD -> Students
- Review student academic status and class structure

4. Mentor and Adviser Mapping
- Go to HOD -> Mentors
- Go to HOD -> Class Advisers
- Ensure each class has an adviser and mentoring structure

5. Course and Notification Oversight
- Go to HOD -> Courses
- Go to HOD -> Notifications
- Ensure communication is active and department updates are published

## Routine Checks
- Verify no class is left without faculty/adviser
- Verify pending academic items are closed
- Verify department announcements are current
