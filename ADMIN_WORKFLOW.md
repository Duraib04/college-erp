# Admin Workflow

This workflow is for system setup and governance tasks.

## Objective
Set up and control academic data flow across departments, faculty, and students.

## Steps

1. Department Setup
- Go to Admin -> Departments
- Add Department Code and Department Name
- Verify department appears in list

2. Faculty Setup
- Go to Admin -> Faculty Management
- Add faculty and map each faculty to the correct department
- Verify faculty count in Departments page

3. Student Setup
- Go to Admin -> Student Management
- Add student and select correct department, year, section
- Verify student count in Departments page

4. HOD Assignment
- Go to Admin -> HOD Assignment
- Assign one faculty as HOD per department
- Confirm HOD shows in Departments page

5. Validation
- Check each department shows correct:
  - HOD
  - Faculty count
  - Student count

## Correction Flow
- Wrong faculty department: Edit in Faculty Management
- Wrong student department: Edit in Student Management
- Wrong HOD: Reassign in HOD Assignment
